# Plintus Profile Editor (Paper.js)

Редактор профилей плинтусов на базе Paper.js с возможностью рисования линий, скруглений и редактирования.

## Особенности

- Рисование линий под углом 90 градусов
- Отображение длины линий в миллиметрах
- Сетка с привязкой (шаг 1 мм)
- Создание скруглений (fillet) между двумя линиями
- Выделение и удаление элементов
- Отображение размеров в виде сносок
- Автосохранение

## Установка

1. Скопируйте папку плагина в `wp-content/plugins/`
2. Установите зависимости:
   ```bash
   cd wp-content/plugins/plintus-profile-editor-paperjs
   npm install
   ```
3. Соберите проект:
   ```bash
   npm run build
   ```
4. Активируйте плагин в WordPress админ-панели

## Разработка

Для разработки с hot-reload:
```bash
npm run dev
```

## Технологии

- React 18
- Paper.js 0.12.17
- Zustand для управления состоянием
- Webpack для сборки

## Использование шорткода

Плагин предоставляет шорткод для вывода редактора на фронтенде:

```
[plintus_editor id="123"]
```

### Параметры шорткода:

- `id` (обязательный) - ID профиля (пост типа `plintus_pjs`)
- `readonly` (необязательный) - Режим только для чтения (`true`/`false`, по умолчанию `false`)

### Примеры использования:

```
[plintus_editor id="123"]
[plintus_editor id="123" readonly="true"]
```

### Примечания:

- Шорткод можно использовать на любой странице или в записи WordPress
- На одной странице можно использовать несколько шорткодов с разными профилями
- Скрипты и стили загружаются автоматически только на страницах, где используется шорткод

## Отличия от Konva версии

Этот плагин использует Paper.js вместо Konva.js для работы с canvas. Paper.js предоставляет более мощные возможности для работы с векторной графикой и путями.
